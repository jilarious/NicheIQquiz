## 🚀 How to Update the Site

### Adding a new file or updating an existing one:
1. Go to [github.com](https://github.com) and open the **nicheiqquiz** repository
2. Click **"Add file"** → **"Upload files"**
3. Drag your new or updated `.html` file onto the upload box
4. Scroll down and click **"Commit changes"**
5. Wait 30 seconds — Netlify auto-deploys instantly ✅

That's it. No drag and drop to Netlify. No manual steps. Just upload to GitHub and the site updates automatically.

## 📧 Email Capture

Emails are collected via **Netlify Forms** — no extra accounts needed.

To view collected emails:
1. Go to [app.netlify.com](https://app.netlify.com)
2. Click your site
3. Click **"Forms"** in the top menu
4. You'll see all form submissions organized by form name

### Forms currently active:
- `pickleball-iq-gate` — email gate before Pickleball IQ results (includes score)
- `pickleball-iq-updates` — "notify me" signup at bottom of Pickleball IQ results
- `pickleball-iq-expert-gate` — email gate before Expert Mode results (includes score)
- `concert-poster-iq-gate` — email gate before Concert Poster IQ results (includes score)
- `concert-poster-updates` — "notify me" signup at bottom of Concert Poster results
